Formula Explanation

Standing waves and organ pipe questions are formula-friendly if boundary conditions are clear.

Wave relation: v = fλString fixed at both ends: fₙ = nv/2LOpen pipe: fₙ = nv/2L, n = 1, 2, 3...Closed pipe: fₙ = nv/4L, n = 1, 3, 5...Open end of pipe: displacement antinodeClosed end of pipe: displacement nodeDistance between two consecutive nodes = λ/2Distance between node and adjacent antinode = λ/4

NEET key: Closed organ pipe has only odd harmonics, while open organ pipe has all harmonics.

Simple NEET Explanation

  • Standing wave forms when two waves of same frequency and amplitude travel in opposite directions.
  • Node is a point of zero displacement.
  • Antinode is a point of maximum displacement.
  • String fixed at both ends has nodes at both ends.
  • Open pipe has antinode at open end.
  • Closed pipe has node at closed end and antinode at open end.
  • Resonance happens when driving frequency matches a natural frequency.

NEET Traps

  • Do not confuse displacement node with pressure node in air columns.
  • Open end is displacement antinode.
  • Closed end is displacement node.
  • Closed pipe supports only odd harmonics.
  • Open pipe supports all harmonics.
  • Distance between adjacent nodes is λ/2.
  • Distance between node and nearby antinode is λ/4.
  • Fundamental frequency of closed pipe is v/4L, not v/2L.
