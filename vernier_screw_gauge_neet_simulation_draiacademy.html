Formula Explanation

Measurement questions are very scoring in NEET if least count and zero error are handled correctly.

Vernier reading = MSR + VCD × LC Correct reading = Observed reading − Zero error Zero correction = − Zero error Screw gauge least count = Pitch / Number of circular scale divisions Screw gauge reading = MSR + CSR × LC Positive zero error is subtracted from observed reading. Negative zero error is added to observed reading.

NEET key: Do not forget units. Vernier often uses cm or mm; screw gauge often uses mm.

Simple NEET Explanation

  • Least count is the smallest measurement an instrument can measure accurately.
  • In vernier caliper, read the main scale reading just before vernier zero.
  • Then find the vernier division that exactly matches a main scale mark.
  • Observed reading = main scale reading + vernier scale reading.
  • In screw gauge, pitch means distance moved by screw in one full rotation.
  • Circular scale reading is multiplied by least count.
  • Zero error must be corrected to get the true reading.

NEET Traps

  • Do not confuse zero error and zero correction.
  • Correct reading = observed reading − zero error.
  • Positive zero error makes observed reading too high.
  • Negative zero error makes observed reading too low.
  • For screw gauge, least count = pitch / circular divisions.
  • Main scale reading should be read just before the zero of vernier/circular scale.
  • Always check whether answer is required in cm, mm, or metre.
  • Significant figures should match instrument precision.

NEET-Style Observation Questions

  1. What is least count and why is it important?
  2. If zero error is positive, should it be added or subtracted?
  3. How do you calculate screw gauge least count?
  4. Why is zero correction equal to negative of zero error?
  5. Which reading is taken first in vernier caliper: main scale or vernier scale?
  6. What happens to accuracy when least count becomes smaller?
  7. Why should unit conversion be checked carefully in measurement questions?
