Formula Explanation

Pulley problems become easy when each block is treated separately using FBD.

Atwood machine: a = (m₂ − m₁)g / (m₁ + m₂) Atwood tension: T = m₁(g + a) = m₂(g − a) Table + hanging mass: a = m₂g / (m₁ + m₂) Rough table: a = (m₂g − μm₁g) / (m₁ + m₂) Friction: f = μN Incline component: m₁g sinθ down the plane Normal on incline: N = m₁g cosθ Ideal string and pulley: tension is same throughout the string

NEET key: Tension is not always equal to weight. It depends on acceleration.

Simple NEET Explanation

  • A pulley changes the direction of tension.
  • For an ideal string and pulley, tension is the same throughout the string.
  • Acceleration of connected blocks has the same magnitude.
  • In Atwood machine, heavier mass moves down and lighter mass moves up.
  • In table-block system, hanging mass pulls the block on the table.
  • Friction opposes the motion or tendency of motion.
  • Always draw FBD separately for each block.
  • Choose positive direction along acceleration to avoid sign mistakes.

NEET Traps

  • Tension is not equal to weight unless acceleration condition allows it.
  • Both connected blocks have same acceleration magnitude in ideal string.
  • Friction acts opposite to motion or tendency of motion.
  • Normal reaction on incline is mg cosθ, not mg.
  • Component along incline is mg sinθ.
  • Do not use one combined equation before understanding directions.
  • If driving force is less than friction limit, system may remain at rest.
  • Massless string and frictionless pulley are ideal assumptions.
