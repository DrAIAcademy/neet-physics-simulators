Formula Explanation

Convex lens is a converging lens. Use Cartesian sign convention carefully in NEET numericals.

Lens formula: 1/f = 1/v − 1/u Magnification: m = v/u = h'/h Power of lens: P = 1/f in metre For convex lens, focal length f is positive. Object distance is usually negative: u = −object distance. Real image: opposite side of lens. Virtual image: same side as object.

NEET key: Convex lens does not always form a real image. If the object is between F and O, image is virtual, erect, and magnified.

Simple NEET Explanation

  • Convex lens is a converging lens.
  • A ray parallel to principal axis passes through focus after refraction.
  • A ray through optical center goes straight without deviation.
  • A ray passing through focus becomes parallel to principal axis after refraction.
  • If object is beyond 2F, image forms between F and 2F and is diminished.
  • If object is at 2F, image forms at 2F and is same size.
  • If object is between F and 2F, image forms beyond 2F and is magnified.
  • If object is at F, image forms at infinity.
  • If object is between F and optical center, image is virtual, erect, and magnified.

NEET Traps

  • Convex lens does not always form a real image.
  • If object is between F and O, image is virtual and erect.
  • Ray through optical center does not bend.
  • Real image is inverted; virtual image is erect.
  • For convex lens, focal length is positive.
  • Use sign convention carefully in numerical problems.
  • Magnification sign tells image orientation.
  • Object at focus gives image at infinity.
  • Object at 2F gives image at 2F with same size.
  • Power formula requires focal length in metre, not centimetre.
